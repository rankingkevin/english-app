Google Sheets 매일 자동 실행 설정
1. Google Sheets → 확장 프로그램 → Apps Script
2. 하단 버튼으로 코드 받기 → 전체 붙여넣기
3. GEMINI_API_KEY 입력
4. 배포 → 새 배포 → 웹 앱 → 액세스: 모든 사용자
5. 배포 URL → 설정 탭 Webhook URL에 입력
6. 트리거 → 시간 기반 → 매일 오전 9시
